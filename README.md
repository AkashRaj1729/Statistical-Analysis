# 📊 Statistical Analysis of Crime in Bihar

A comprehensive statistical study of crime trends in Bihar using quantitative methods, regression analysis, hypothesis testing, and data visualization techniques implemented in R.

---

## 📖 Project Overview

This project investigates crime patterns and trends in the state of Bihar through statistical analysis and data-driven modeling. The study focuses on understanding the relationship between time, crime incidence, and policy interventions using various statistical techniques.

The project was completed as part of undergraduate coursework in Statistics and Data Analysis.

---

## 🎯 Objectives

- Analyze trends in crimes against women in Bihar.
- Study murder and violent crime statistics.
- Investigate the effectiveness of government interventions.
- Apply statistical hypothesis testing to real-world data.
- Develop predictive and explanatory statistical models.
- Visualize crime trends using graphical techniques.
